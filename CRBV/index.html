OFFICIAL PAGE

What is "Funkin' Corruption Reimagined"?

Funkin' Corruption Reimagined is a reimagined version of the youtuber DyanStella, based on the mod created by the youtuber Phantom Fear, which is also a mod of the popular Newgrounds game: "Friday Night Funkin". Despite being similar to Phantom Fear's mod, this version completely changes the story and how events unfold, making it a completely new and interesting experience.

What is "Funkin' Corruption Reimagined: BV"?

Funkin' Corruption Reimagined: BV (Bruxter Version) is a modification of the mentioned mod. This version changes the original FCR by adding new animations, events, cinematics (made by me), and aesthetic changes to some of the characters. The mod was completed on June 3, 2023, and there is currently a YouTube video with gameplay of the entire game, you can watch it in my youtube channel, which is linked at the end of the page.

Synopsis

After the events occured in FNF Corruption, a mysterious entity travels back in time to the beginning of the story, searching for a way to fix the tragic events that occurred.
Will this entity make a change and prevent the catastrophe?
